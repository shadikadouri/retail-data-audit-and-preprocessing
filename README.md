# Online Retail Data Merger 🛒

This repository contains a Python script designed to merge and prepare a large two-year historical dataset (2009-2011) from an online retail company. 

## Objective
The goal of this mini-project is to efficiently concatenate two massive CSV files into a single, clean, and queryable dataset using the `pandas` library, paving the way for exploratory data analysis (EDA) and predictive modeling.

## Technologies Used
* **Python 3.x**
* **Pandas**: For fast data manipulation and concatenation.

## Data Source
The raw data consists of two files containing transactional data for an online retail store:
1. `online_retail_II - Year 2009-2010.csv`
2. `online_retail_II - Year 2010-2011.csv`

*(Note: The datasets are not uploaded in this repository due to size constraints. You can find the original dataset on platforms like Kaggle or UCI Machine Learning Repository).*
